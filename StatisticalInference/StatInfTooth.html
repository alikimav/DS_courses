library(datasets) data(“ToothGrowth”)

Data set description:

The ToothGrowth data set investigates the effect of vitamin C on tooth growth in guinea pigs. The data set contains 60 observations on 3 variables: len denotes the tooth length, supp represents the vitamin C supplement type [absorbic acid (VC) or orange juice (OJ)], and dose is the dose in milligrams. Note: I am inferring that “VC” represents absorbic acid as the dataset description is poor at best.

The response is the tooth length and we are interested in exploring whether growth is affected by dose or supplement type or both factors. We achieve this through statistical inference; however, it is of interest to complete an exploratory data analysis first in order to drive our inferential statistics.

We first reshape the data set such that we obtain the average tooth length according to supplement type and dose.

library(reshape2)
mt <- melt(ToothGrowth, id = c("supp", "dose"), measure.vars = c("len"))
aveLen <- dcast(mt, supp + dose ~ variable, mean)
aveLen

A first glance at the data indicates that an increase in dose leads to an increase in tooth length while OJ as the supplement type leads to higher tooth length at 0.5, 1 mg dose but the effect decreases significantly at 0.2 mg. This is most clearly observed graphically. We start with a plot depicting the mean tooth length with dose grouped by supplement which shows an increasing trend for both supplements as dose increases.
